14 /*!***************************************************************************
15  * @defgroup argusdca Dynamic Configuration Adaption
16  * @ingroup argusapi
17  *
18  * @brief Dynamic Configuration Adaption (DCA) parameter definitions and API functions.
19  *
20  * @details The DCA contains an algorithms that detect ambient conditions
21  * and adopt the device configuration to the changing parameters
22  * dynamically while operating the sensor. This is achieved by
23  * rating the currently received signal quality and changing the
24  * device configuration accordingly to the gathered information
25  * from the current measurement frame results before the next
26  * integration cycle starts.
27  *
28  * The DCA consists of the following features:
29  * - Static or Dynamic mode. The first is utilizing the nominal
30  * values while the latter is dynamically adopting between min.
31  * and max. value and starting from the nominal values.
32  * - Analog Integration Depth Adaption (from multiple patterns down to single pulses)
33  * - Optical Output Power Adaption
34  * - Pixel Input Gain Adaption (w/ ambient light rejection)
35  * - ADC Sensitivity (i.e. ADC Range) Adaption
36  * - Power Saving Ratio (to decrease the average output power and thus the current consumption)
37  * - All that features are heeding the Laser Safety limits.
38  * .
39  *
40  * @addtogroup argusdca
41  * @{
42  *****************************************************************************/
