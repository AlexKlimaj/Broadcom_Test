AFBR-S50 API Example Projects
=============================

- AFBR_S50_Example_KL46z: 
	Basic examples to demonstrate the usage of the AFBR-S50 SDK; Can be run on the AFBR-S50 Evaluation Board containing the NXP MKL46z MCU. Import the project into MCUXpressoIDE.
	
- AFBR_S50_Example_KL17z: 
	Basic examples to demonstrate the usage of the AFBR-S50 SDK; Can be run on the AFBR-S50 Reference Board containing the NXP MKL17z MCU. Import the project into MCUXpressoIDE.
	
- AFBR_S50_ExplorerApp_KL46z: 
	The AFBR-S50 SDK embedded into the evaluation board software containing a serial communication interface (USB or UART); Can be run on the AFBR-S50 Evaluation Board containing the NXP MKL46z MCU. Import the project into MCUXpressoIDE.

- AFBR_S50_ExplorerApp_KL17z: 
	The AFBR-S50 SDK embedded into the evaluation board software containing a serial communication interface (USB or UART); Can be run on the AFBR-S50 Reference Board containing the NXP MKL17z MCU. Import the project into MCUXpressoIDE.

.

Please refer the API Reference Manual on how to compile and run the example projects.
